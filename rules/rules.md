# Kei Project — Hard Rules

> These are non-negotiable. Never violate without explicit instruction from Keyvin.

## Stack Rules
- **Plain HTML / CSS / JS only** — zero frameworks, zero build steps, zero dependencies
- No React, No Vue, No Tailwind, No bundler, No npm
- No build step — if it requires compiling or bundling, it's wrong for this project
- Preview by opening `index.html` directly in browser

## Design Rules
- Background: `#0F0F23` (dark navy) — do not change
- Primary accent: `#7C3AED` (electric violet) — do not change
- CTA / highlight: `#F43F5E` (rose red) — do not change
- Aesthetic: dark neon cyberpunk — all additions must fit this aesthetic

## Content Rules
- **Do not reintroduce "Vibe Coder"** — removed by decision, stays removed
- **Do not reintroduce the tech stack icon cloud** — removed by decision, stays removed
- Do not hardcode form submissions to a placeholder endpoint — form backend is pending

## Code Rules
- Do not add frameworks even if it "would be easier"
- Do not introduce npm or package.json for any reason
- All JS goes in `js/`, all CSS goes in `css/`, all images in `assets/`
- Never commit without Keyvin's explicit instruction
